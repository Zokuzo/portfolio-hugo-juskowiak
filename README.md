## Hugo Juskowiak — Portfolio

A modern, fast portfolio built with Next.js, TypeScript, Tailwind CSS v4, and shadcn/ui components. Animations powered by Framer Motion.

### Preview

- Development: `http://localhost:3000`
- Production: `https://portfolio-hugo-juskowiak.vercel.app/`
- GitHub: `https://github.com/Zokuzo`
- LinkedIn: `https://www.linkedin.com/in/hugo-juskowiak-991900192/`

### Tech Stack

- Next.js 16, React 19, TypeScript
- Tailwind CSS v4, shadcn/ui
- Framer Motion, Lucide Icons

### Getting Started

```bash
# install (uses npm here; pnpm/yarn also work)
npm install --legacy-peer-deps

# dev
npm run dev

# production build
npm run build
npm start
```

### Scripts

- `dev`: start Next.js in development
- `build`: production build
- `start`: run the built app
- `lint`: run eslint (if configured)

### Project Structure

- `app/` — Next.js app router pages and layout
- `components/` — UI and section components
- `hooks/` — shared React hooks
- `lib/` — utilities
- `public/` — static assets

### Customize

- Colors and theme tokens: `app/globals.css`
- Sections: edit files in `components/` (e.g., `hero-section.tsx`, `projects-section.tsx`, etc.)
- Social/contact links: `components/hero-section.tsx`, `components/contact-section.tsx`
